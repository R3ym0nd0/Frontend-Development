# 🖼️ Images

## Basic Image Example:
    <img src="image.jpg" alt="Description" width="300" height="200">

- `Image Tag <img>` — **inserts an image**
## Image Attributes:

- `src` — **image file path or URL**
- `alt` — **text shown if image fails or for screen readers**
- `width / height` — **size of the image (pixels)**
